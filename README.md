# disgo

The Go implementation of the Dispatch protocol

## To install

1. Install [Go](https://golang.org/)
2. `go get github.com/DispatchLabs/disgo`
3. `cd $GOPATH/src/github.com/DispatchLabs/disgo`
4. `go get`

## To run

1. cd $GOPATH/bin
2. disgo is your executable binary
3. run ```./disgo daemon```
